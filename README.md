# Airline Ticketing System — C++

Computer Programming Lab project. CLI-based airline reservation system
for national and international flights, built in C++.

## Features
- Search national and international flights
- Book tickets for multiple passengers (name, CNIC, contact)
- Auto-generates boarding pass with seat and gate number
- Cancel booking by CNIC
- Menu-driven CLI interface

## Flight Routes
**National:** KHI-ISB, ISB-LHR, LHR-KHI, KHI-SKD, KHI-HYD
**International:** KHI-DXB, KHI-LND, KHI-NYK, KHI-BKK, KHI-FRA

## Concepts Used
- Structs (Passenger data)
- Vectors (dynamic passenger list)
- Functions (SearchFlight, BookFlight, CancelFlight)
- String handling with getline
